# Cinemax Frontend

Frontend en React + Vite + Tailwind construyendo el práctico 4 de Cinemax

## Cómo usarlo rápido

1. Levantá el backend:

```bash
npm install
npm run start:dev
```

2. Instalá y levantá el frontend:

```bash
npm install
npm run dev
```

3. Abrí `http://localhost:5173`.

La variable de entorno está en `.env`:

```env
VITE_API_URL=http://localhost:3000/api
```
